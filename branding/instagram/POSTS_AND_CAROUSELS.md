# Instagram Posts & Carousels — Stop & Think

> 12+ post template sẵn dùng cho Instagram. Mỗi template có:
> 1. Visual spec (kích thước, layout, colors)
> 2. Caption đề xuất
> 3. Hashtag set
> 4. Khi nào post

---

## 🎨 Quy tắc visual chung

### Tools đề xuất
- **Canva** — preset 1080×1080 + 1080×1350 + 1080×1920
- **Figma** — nếu muốn control nhiều hơn
- **Procreate / iPad** — cho hand-drawn elements

### Always include
- Logo "Stop & Think" + dot copper ở 1 góc (top-right hoặc bottom-right)
- Domain "stopandthink.pages.dev" cuối post (hoặc slide cuối carousel)
- 1 màu palette (rust hoặc copper hoặc navy) — không cả 3 trong 1 post

### Font hierarchy trên ảnh
- **Headline:** Fraunces serif 60-90px weight 500-700
- **Body:** Fraunces serif 28-36px weight 400
- **Label/meta:** JetBrains Mono UPPERCASE 12-16px letter-spacing 0.2em
- **CTA:** Inter sans-serif 14px UPPERCASE letter-spacing 0.15em

## 📋 Checklist trước khi post 1 carousel

- [ ] Slide 1 có hook 8 từ đầu hoặc visual mạnh
- [ ] Mỗi slide có 1 ý — không nhồi
- [ ] Slide cuối có CTA rõ (link bio / comment / save)
- [ ] Visual nhất quán — cùng palette + font
- [ ] Caption hỗ trợ slide, không lặp lại
- [ ] Hashtag set hợp với chủ đề
- [ ] Đã preview trên mobile (1080 trên màn hình to khác trên IG)

## 🚀 Posting flow

1. Chọn template từ file này
2. Chỉnh visual trong Canva — pick template duy nhất 1 lần, save làm Brand Kit
3. Copy caption — paste vào IG
4. Pick hashtag set từ [`HASHTAGS.md`](./HASHTAGS.md)
5. Schedule (Buffer / Later) hoặc post manual
6. Trong 30 phút sau post — reply comment đầu tiên (boost reach)
7. Story share post — tăng visibility
