## 📋 Day 1 quick checklist

```
□ Export profile-picture.svg → PNG 500×500 (cloudconvert.com)
□ Upload PNG làm profile picture trên IG
□ Edit profile → Name field: "Stop & Think · Quang Ton"
□ Edit profile → Bio: paste Variant 1 từ BIO.md
□ Edit profile → Verify link "stop-and-think.pages.dev"
□ Settings → Account → Switch to Professional Account → "Personal Blog"
□ Save tất cả
```

## 🎨 Quick visual reference

**Color palette dùng trong tất cả post:**
- Cream `#f5ede0` (background chính)
- Rust `#8b3a2b` (accent)
- Copper `#c4923c` (accent ấm)
- Navy `#2d4a5c` (alternative dark)
- Ink `#1a1410` (text/border)

**Font:**
- Fraunces serif — heading + quote
- JetBrains Mono — labels (UPPERCASE letterspacing 0.2em)

**Tools đề xuất:**
- Canva (free, có template) — visual chính
- CloudConvert.com (free) — SVG → PNG
- Buffer / Later (free tier) — schedule post
